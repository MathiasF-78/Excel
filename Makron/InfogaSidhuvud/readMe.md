## InfogaSidhuvud Macro

### Beskrivning
Detta makro används för att infoga text i sidhuvudet på det aktiva arket i Excel.

### Användning
1. Kör makrot "InfogaSidhuvud".
2. Ange den önskade texten i den dialogruta som visas.

### Åtgärder som utförs av makrot:
- Ett dialogruta visas där användaren ombeds att ange texten som ska infogas i sidhuvudet.
- Den angivna texten infogas i sidhuvudet på det aktiva arket.

### Anmärkningar
- Endast texten som anges i dialogrutan kommer att infogas i sidhuvudet.
- Sidhuvudet kan innehålla text, datum, filnamn eller annan information.
- Om sidhuvudet redan innehåller text, kommer den nya texten att ersätta den befintliga texten.
- Makrot antar att texten ska placeras i centrum av sidhuvudet.
- Felhantering är inte implementerad i detta makro.
