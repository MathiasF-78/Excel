## DeleteBlankWorksheets Macro

### Beskrivning
Detta makro tar bort alla tomma kalkylblad i den aktiva arbetsboken.

### Användning
1. Kör makrot "DeleteBlankWorksheets".
2. Alla tomma kalkylblad i den aktiva arbetsboken kommer att tas bort.

### Anmärkningar
- Om ett kalkylblad inte innehåller några celler med data (dvs. tomma celler) tas det bort av makrot.
- Om du vill använda detta makro bör du se till att ha en säkerhetskopia av din arbetsbok, eftersom det inte finns något sätt att ångra borttagningen av kalkylblad.
