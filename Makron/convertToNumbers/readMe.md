## ConvertTextToNumber Macro

### Beskrivning
Detta makro konverterar markerad text till numeriskt format med noll decimaler i Excel. Det är användbart när du behöver behandla numeriska värden istället för text och vill undvika fel som kan uppstå på grund av textformat.

### Användning
1. Markera de celler i Excel som innehåller text som du vill konvertera till numeriskt format.
2. Kör makrot "ConvertTextToNumber".
3. Makrot konverterar texten till numeriskt format med noll decimaler i de markerade cellerna.

### Anmärkningar
- Makrot kommer att ändra formatet för de markerade cellerna till numeriskt med noll decimaler och ersätta texten med motsvarande numeriska värden.
- Se till att markera de önskade cellerna innan du kör makrot eftersom det endast påverkar de markerade cellerna.
- Observera att om någon av de markerade cellerna innehåller icke-numerisk text kan det resultera i ett fel vid konverteringen.
