## AddSuffix Macro

### Beskrivning
Detta makro lägger till en suffix-sträng efter varje cell i det markerade cellområdet.

### Användning
1. Markera det cellområde där du vill lägga till suffixet.
2. Kör makrot.
3. En dialogruta visas där du kan ange den önskade texten för suffixet.
4. Skriv in den önskade texten och klicka på OK.

### Anmärkningar
- Om du lämnar dialogrutan tom och klickar på OK kommer inget suffix att läggas till.
- Suffixet läggs till efter texten i varje cell i det markerade cellområdet.
