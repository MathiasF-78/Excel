## tusenKr Macro

### Beskrivning
Detta makro formaterar det markerade området så att det visas i tusentals kronor-format.

### Användning
1. Markera det område i Excel som du vill formatera till tusentals kronor-format.
2. Kör makrot "tusenKr".
3. Det markerade området kommer nu att visas med tusentals separatorer och suffixet "kr".

### Anmärkningar
- Makrot ändrar bara formatet för de markerade cellerna till tusentals kronor-format med standardtusentalsseparator (#,###) och suffixet "kr".
- Om du vill applicera detta format på en specifik kolumn eller rad, markera först den önskade kolumnen eller raden innan du kör makrot.
