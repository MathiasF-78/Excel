## MarkeraTommaCeller Macro

### Beskrivning
Detta makro används för att markera alla tomma celler inom det markerade området med en specifik färg. Standardfärgen är ljusgrön.

### Användning
1. Kör makrot "MarkeraTommaCeller".
2. Alla tomma celler inom det markerade området kommer att markeras med den valda färgen.

### Åtgärder som utförs av makrot:
- Makrot hittar alla tomma celler inom det markerade området.
- Det ändrar bakgrundsfärgen på dessa tomma celler till den valda färgen (ljusgrön som standard).

### Anmärkningar
- Du kan justera färgen genom att ändra ColorIndex-värdet i makrokoden enligt dina önskemål.
- Om det markerade området inte innehåller några tomma celler kommer ingen färgändring att göras.
