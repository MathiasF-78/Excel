## CloseAllWorkbooks Macro

### Beskrivning
Detta makro stänger alla öppna arbetsböcker i Excel. Det sparar eventuella ändringar innan arbetsböckerna stängs.

### Användning
1. Kör makrot "CloseAllWorkbooks".
2. Alla öppna arbetsböcker stängs automatiskt och eventuella ändringar sparas.

### Anmärkningar
- Se till att spara eventuella ändringar innan du kör makrot eftersom det kommer att stänga alla arbetsböcker utan att fråga om du vill spara ändringarna.
- Om du vill undvika att spara ändringar automatiskt innan du stänger arbetsböckerna kan du ändra parameterinställningen "SaveChanges" till "False" i koden.
