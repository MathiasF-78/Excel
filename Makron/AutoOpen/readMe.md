## AutoOpen Macro

### Beskrivning
Detta makro visar ett välkomstmeddelande när arbetsboken öppnas. Det använder händelsen auto_open, vilket innebär att det utförs automatiskt när arbetsboken öppnas.

### Användning
- När arbetsboken öppnas visas ett välkomstmeddelande med instruktioner till användaren.

### Anmärkningar
- För att använda detta makro behöver du inte göra några manuella åtgärder. Det räcker med att namnge makrot "auto_open" och spara arbetsboken.
- Meddelandet som visas kan anpassas genom att ändra texten i makrokoden.
