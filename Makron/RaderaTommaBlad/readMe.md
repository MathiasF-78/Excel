## RaderaTommaBlad Macro

### Beskrivning
Detta makro tar bort alla tomma blad (ark) i den aktuella arbetsboken.

### Användning
1. Kör makrot "RaderaTommaBlad".

### Anmärkningar
- Makrot förutsätter att det finns flera blad (ark) i den aktuella arbetsboken.
- Endast tomma blad (ark) kommer att påverkas av makrot. Ett tomt blad definieras som ett blad där inga celler har något innehåll.
- När makrot körs, kommer alla tomma blad (ark) att tas bort från arbetsboken utan att någon bekräftelse krävs från användaren.
- Det är viktigt att notera att makrot inte tar bort blad (ark) som innehåller bara formatering eller objekt som bilder eller diagram utan innehåll i cellerna.
