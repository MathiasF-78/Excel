## OpenWorkbookAsAttachment Macro

### Beskrivning
Detta makro öppnar den aktiva arbetsboken som en bifogad fil i ett e-postmeddelande.

### Användning
1. Kör makrot "OpenWorkbookAsAttachment".
2. Ett fönster för att skicka e-post öppnas med den aktiva arbetsboken som bifogad fil.

### Anmärkningar
- Makrot förutsätter att det finns en e-postklient (till exempel Microsoft Outlook) konfigurerad på datorn.
- När du kör makrot visas e-postklientens dialogruta för att skapa ett nytt meddelande med den aktiva arbetsboken som en bifogad fil.
