## MarkeraDubletter Macro

### Beskrivning
Detta makro används för att markera dubletter inom den markerade cellområdet med grön fyllning.

### Användning
1. Markera det cellområde där du vill kontrollera dubletter.
2. Kör makrot "MarkeraDubletter".
3. Dubletter inom det markerade cellområdet kommer att markeras med grön fyllning.

### Åtgärder som utförs av makrot:
- Makrot loopar igenom det markerade cellområdet.
- Om en cell innehåller en text som redan finns i det markerade området, markeras den med grön fyllning.

### Anmärkningar
- Detta makro fungerar endast för att markera dubletter i textformat.
- Dubletterna markeras med grön fyllning för att göra dem synliga, men du kan ändra färgen genom att justera koden enligt dina preferenser.
