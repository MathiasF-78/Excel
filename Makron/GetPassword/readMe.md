## GetPassword Macro

### Beskrivning
Detta makro används för att avslöja lösenord som skyddar en Excel-kalkylbladscell. Det testar alla möjliga kombinationer av tecken tills lösenordet hittas och kalkylbladet blir upplåst.

### Användning
1. Öppna Excel-dokumentet som innehåller det skyddade kalkylbladet.
2. Kör makrot "GetPass".
3. Om kalkylbladet är skyddat, kommer makrot att försöka hitta lösenordet genom att testa alla möjliga kombinationer av tecken.
4. När lösenordet har hittats och kalkylbladet är upplåst, visas ett meddelande som anger att processen är klar.

### Anmärkningar
- Detta makro är endast avsett för att användas på dina egna dokument eller med tillstånd från dokumentets ägare.
- Användningen av detta makro på dokument som du inte äger kan vara i strid med integriteten och sekretessen hos dokumentets innehåll.
- Det kan ta lång tid att testa alla möjliga kombinationer av tecken, särskilt om lösenordet är långt eller komplext.
- Användning av detta makro kan vara olagligt eller strida mot användaravtalet för vissa dokument.
- Försök inte använda detta makro för att bryta lösenord på dokument som du inte har rätt att få tillgång till.
