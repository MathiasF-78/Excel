## SepareraCeller Macro

### Beskrivning
Detta makro separerar redan sammanfogade celler på det aktiva kalkylarket i Excel.

### Användning
1. Markera de sammanfogade celler som du vill separera.
2. Kör makrot "SepareraCeller".
3. De markerade cellerna kommer att separeras och innehållet återställs till ursprungliga positioner.

### Anmärkningar
- Makrot fungerar genom att ta bort sammanfogningen från de markerade cellerna, vilket gör det möjligt att återställa varje cell till dess ursprungliga storlek och innehåll.
- Det är viktigt att notera att om de sammanfogade cellerna innehåller data som inte kan delas upp på ett meningsfullt sätt, kan detta leda till förlust av data när cellerna separeras.
- Använd makrot med försiktighet och se till att data i de sammanfogade cellerna kan återställas till sina ursprungliga positioner utan att förlora viktig information.
