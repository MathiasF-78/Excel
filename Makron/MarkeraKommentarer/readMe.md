## MarkeraKommentarer Macro

### Beskrivning
Detta makro används för att markera alla celler med kommentarer i arket med en specifik färg. Standardfärgen är gult för bakgrund och svart för text.

### Användning
1. Kör makrot "MarkeraKommentarer".
2. Alla celler med kommentarer kommer att markeras med den valda färgen.

### Åtgärder som utförs av makrot:
- Makrot hittar alla celler som innehåller kommentarer.
- Det ändrar bakgrundsfärgen på dessa celler till gult och textfärgen till svart som standard.

### Anmärkningar
- Du kan justera färgerna genom att ändra RGB-värdena i makrokoden enligt dina önskemål.
- Detta makro fungerar endast för att markera celler med kommentarer, det ändrar inte själva kommentarerna.
