## AddPrefix Macro

### Beskrivning
Detta makro lägger till en prefix-sträng före varje cell i det markerade cellområdet.

### Användning
1. Markera det cellområde där du vill lägga till prefixet.
2. Kör makrot.
3. En dialogruta visas där du kan ange den önskade texten för prefixet.
4. Skriv in den önskade texten och klicka på OK.

### Anmärkningar
- Om du lämnar dialogrutan tom och klickar på OK kommer inget prefix att läggas till.
- Prefixet läggs till före texten i varje cell i det markerade cellområdet.
