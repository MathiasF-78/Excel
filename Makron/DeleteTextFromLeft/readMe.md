## Delete_Text_From_Left Macro

### Beskrivning
Detta makro tar bort ett visst antal tecken från vänster i varje cell i den angivna cellområdet.

### Användning
1. Anropa makrot "Delete_Text_From_Left" och ange det cellområde du vill bearbeta samt det antal tecken du vill ta bort från vänster i varje cell.
   Till exempel kan du använda följande syntax för att ta bort de två första tecknen från vänster i varje cell i cellområdet A1:A10:
