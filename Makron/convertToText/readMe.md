## convertToNumbers Macro

### Beskrivning
Detta makro konverterar valda celler till numeriskt format i Excel.

### Användning
1. Markera de celler som du vill konvertera till numeriskt format.
2. Kör makrot.
3. De markerade cellerna kommer att konverteras till numeriskt format.

### Exempel
Om du har celler med textvärden markerade och kör makrot, kommer texten att konverteras till numeriska värden om möjligt.

### Anmärkningar
- Makrot tillämpar "General" nummerformat på de valda cellerna och ersätter sedan värdet med sig själv för att tvinga Excel att tolka dem som numeriska värden.
- Se till att markera de celler som innehåller textvärden som du vill konvertera innan du kör makrot.
