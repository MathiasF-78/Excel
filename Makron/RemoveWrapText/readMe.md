## RemoveWrapText Macro

### Beskrivning
Detta makro tar bort radbrytning från alla celler i det aktiva arket och justerar sedan automatiskt höjden på raderna och bredden på kolumnerna för att visa allt innehåll utan att behöva radbrytning.

### Användning
1. Kör makrot "RemoveWrapText".

### Anmärkningar
- Makrot tar bort radbrytning från alla celler i det aktiva arket, vilket gör att allt innehåll visas i en enda rad i varje cell.
- Efter att radbrytning har tagits bort justeras automatiskt höjden på raderna och bredden på kolumnerna så att allt innehåll syns utan att behöva använda radbrytning.
- Detta kan vara användbart för att snabbt anpassa storleken på celler för att visa allt innehåll tydligt.
