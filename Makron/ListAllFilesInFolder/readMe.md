## ListAllFilesInFolder Macro

### Beskrivning
Detta makro används för att lista alla filer som finns i en angiven mapp och dess undermappar i Excel.

### Användning
1. Kör makrot "ListAllFilesInFolder".
2. En dialogruta kommer att visas där du kan välja den önskade mappen. Välj mappen och klicka på "Öppna".

### Åtgärder som utförs av makrot:
- Makrot öppnar en dialogruta för att välja en mapp.
- Det listar sedan alla filer som finns i den valda mappen och dess undermappar.
- Filerna listas som hyperlänkar i den aktiva kalkylbladet i Excel.

### Anmärkningar
- Det är viktigt att välja en mapp där du har behörighet att komma åt dess innehåll.
- Om mappen innehåller många filer kan det ta lite tid för makrot att lista dem alla.
- Om du vill ändra var filerna listas kan du justera koden för att passa dina behov.
