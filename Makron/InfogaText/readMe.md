## InfogaText Macro

### Beskrivning
Detta makro används för att lägga till valfri text till höger om det befintliga innehållet i de markerade cellerna i Excel.

### Användning
1. Markera de celler där du vill lägga till text.
2. Kör makrot "InfogaText".
3. Ange den text som du vill lägga till i de markerade cellerna i dialogrutan som visas.

### Åtgärder som utförs av makrot:
- Makrot visar en dialogruta där användaren ombeds att ange den text som ska läggas till till höger om det befintliga innehållet.
- Den angivna texten läggs till till höger om det befintliga innehållet i varje markerad cell.

### Anmärkningar
- Endast de markerade cellerna kommer att modifieras av makrot.
- Den angivna texten kommer att läggas till till höger om det befintliga innehållet i varje cell.
- Om en cell är tom kommer den nya texten att läggas till direkt.
- Makrot antar att texten ska läggas till till höger om det befintliga innehållet utan något mellanrum.
