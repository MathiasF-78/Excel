## Speak Macro

### Beskrivning
Detta makro använder Text-till-tal-funktionen i Excel för att läsa upp texten som finns i det markerade området, cell för cell.

### Användning
1. Markera det område i Excel där du vill att texten ska läsas upp.
2. Kör makrot "Speak".
3. Excel kommer att läsa upp texten, en cell åt gången, från det markerade området.

### Anmärkningar
- Makrot använder Text-till-tal-funktionen i Excel för att läsa upp texten i det markerade området.
- Texten läses upp en cell i taget, från vänster till höger och uppifrån och ned i det markerade området.
- Se till att markera endast de celler som innehåller den text som du vill att Excel ska läsa upp.
