## Speak Macro

### Beskrivning
Detta makro läser upp en användares inmatade text med hjälp av Text-till-tal-funktionen i Windows.

### Användning
1. Kör makrot "Speak".
2. En dialogruta visas där användaren ombeds att ange texten som ska läsas upp.
3. Skriv in önskad text och klicka på OK.
4. Makrot kommer att läsa upp den angivna texten med hjälp av Text-till-tal-funktionen.

### Anmärkningar
- Makrot använder Windows-röstigenkänning (sapi.spvoice) för att läsa upp den angivna texten.
- När du kör makrot öppnas en dialogruta där användaren kan ange texten. Om inget text anges kommer inget att läsas upp.
- Se till att ha högtalare eller hörlurar anslutna till datorn för att höra den upplästa texten.
