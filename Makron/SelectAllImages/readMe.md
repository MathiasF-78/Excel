## SelectAllPics Macro

### Beskrivning
Detta makro markerar alla bilder (inbäddade eller länkade) på det aktiva kalkylarket i Excel.

### Användning
1. Kör makrot "SelectAllPics".
2. Alla bilder på det aktiva kalkylarket kommer att markeras.

### Anmärkningar
- Makrot fungerar endast om det finns bilder på det aktiva kalkylarket. Om det inte finns några bilder kommer makrot inte att göra någonting.
- Om det finns både inbäddade och länkade bilder kommer båda typerna att markeras när makrot körs.
