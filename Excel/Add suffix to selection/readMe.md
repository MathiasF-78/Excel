## AddSuffix Macro

### Beskrivning
Detta makro lägger till en suffix-sträng till varje cell i den valda cellområdet.

### Användning
1. Markera det cellområde där du vill lägga till suffixet.
2. Kör makrot.
3. Ange den text du vill lägga till som suffix i den dialogruta som visas.
4. Klicka på OK.

### Exempel
Om du markerar cellerna A1:A10 och anger "suffix" som suffix kommer texten i varje cell att följas av "suffix".

### Anmärkningar
- Om du inte anger någon text i dialogrutan kommer makrot inte att göra några ändringar.
- Felkontroller är implementerade för att hantera situationer där cellområdet är tomt eller om fel inträffar under körningen av makrot.
