## AddDegreeSymbol Macro

### Beskrivning
Detta makro lägger till ett gradtecken (°) efter texten i de markerade cellerna i Excel.

### Användning
1. Markera de celler där du vill lägga till gradtecknet.
2. Kör makrot.
3. Gradtecknet läggs till efter texten i varje markerad cell.

### Anmärkningar
- Endast om cellens innehåll är numeriskt kommer gradtecknet att läggas till efter texten.
- Om cellen redan innehåller ett gradtecken läggs ett ytterligare gradtecken till.
