## AddMultipleSheets Macro

### Beskrivning
Detta makro lägger till det önskade antalet nya kalkylblad i arbetsboken.

### Användning
1. Kör makrot.
2. En dialogruta visas där du kan ange det önskade antalet nya kalkylblad som ska läggas till.
3. Ange antalet nya kalkylblad och klicka på OK.

### Anmärkningar
- Om du lämnar dialogrutan tom eller klickar på Avbryt kommer inga nya kalkylblad att läggas till.
- Om du anger ett icke-numeriskt värde i dialogrutan visas ett felmeddelande.
- Om du anger ett negativt antal eller noll i dialogrutan kommer inga nya kalkylblad att läggas till.
